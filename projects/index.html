techinterview.space#

A non-commercial platform for the Kazakhstan IT community. Collects and aggregates salary data (since 2024), hosts company reviews (since 2025), and provides a tool for documenting technical interview feedback (since 2021). The mission is to promote openness and transparency in Kazakhstan’s IT sphere.

  • For whom: IT professionals, job seekers, interviewers, and HR specialists in Kazakhstan
  • Link: techinterview.space
  • Tech stack: Angular, .NET, PostgreSQL, DigitalOcean
  • Vibecoding used: Before 2026 no, since 2026 yes

Repolyze (cli)#

A Rust-based CLI and TUI tool for analyzing local Git repositories. Provides contributor statistics (commits, lines modified, files touched), GitHub-style activity heatmaps, multi-repository batch processing, and SQLite-backed smart caching. Supports JSON and Markdown export. Open source under MIT License.

  • For whom: Team leads, tech leads, and individual contributors who want insights into repository activity and contribution patterns
  • Link: repolyze.app
  • Tech stack: Rust, SQLite, macOS & Linux
  • Vibecoding used: Yes

Claude Community Kazakhstan (website)#

A community platform for AI enthusiasts, developers, and researchers in Kazakhstan who build with Claude. Facilitates knowledge sharing, collaborative projects, and learning about AI capabilities. Supports Russian, English, and Kazakh languages. Open source on GitHub.

  • For whom: AI developers, enthusiasts, and researchers in Kazakhstan
  • Link: claude-community.kz
  • Tech stack: Astro, Google Analytics
  • Vibecoding used: Yes

EV Charge Tracker (iOS)#

An iOS app for tracking electric vehicle charging sessions and expenses. Monitors cost per kilometer, provides 6-month expense statistics with charts, and includes maintenance planning with reminders. All data is stored locally on the device — no data collection or sharing. Open source under MIT License.

  • For whom: Electric vehicle owners who want to track charging costs and driving efficiency
  • Link: App Store, Landing page
  • Tech stack: SwiftUI, Swift, iOS 18+
  • Vibecoding used: Yes

Journey Wallet (iOS)#

An iOS app for organizing travel plans, bookings, documents, and expenses in one place. Supports multi-currency expense tracking (11 currencies), share extension for importing from Mail and Safari, built-in PDF and image viewers, global search, and 6 languages. All data is stored locally on the device. Open source under MIT License.

  • For whom: Travelers who want to keep all journey details, bookings, and documents organized
  • Link: App Store, Landing page
  • Tech stack: SwiftUI, Swift, iOS 18+
  • Vibecoding used: Yes

Creativity Hub (iOS)#

An iOS app for managing creative projects in one place. Capture ideas with automatic source detection (Instagram, TikTok, Pinterest, YouTube), plan with checklists, track expenses with multi-currency support, store documents, set reminders, and log work progress. All data is stored locally on the device. Open source under MIT License.

  • For whom: Creators who want to organize ideas, plan projects, and track progress
  • Link: App Store, Landing page
  • Tech stack: SwiftUI, Swift, iOS 18+
  • Vibecoding used: Yes

Slothy Terminal (macos)#

A GPU-accelerated macOS terminal for AI-assisted coding. Built on libghostty with Metal rendering, it combines a high-performance terminal engine with workspace-aware tabs, automation sidebars, and native AI chat sessions. Includes a built-in Git client with visual revision graphs and integration with Claude CLI or OpenCode. Open source under MIT License.

  • For whom: Developers on Apple Silicon Macs who want AI tools integrated into their terminal workflow
  • Link: Landing page
  • Tech stack: Swift, Zig, libghostty, Metal, macOS 15.0+
  • Vibecoding used: Yes
